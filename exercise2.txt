2. What are the three Boolean operators?

and, or, not.

3. Write out the truth tables of each Boolean operator (that is, every possible combination of Boolean values for the operator and what they evaluate to).

 and

True and False = False

False and True = False

False and False = False

 or
True and True = True

True and False = True

False and True = True

False and False = False

 not
not True = False

not False = True

6. What is the difference between the equal to operator and the assignment operator?

O operador == (igual a) pergunta se dois valores são iguais.
O operador = (atribuição) coloca o valor à direita na variável à esquerda.

7. Explain what a condition is and where you would use one.

Condição é apenas um nome mais específico no contexto das instruções de controle de fluxo. As condições sempre são avaliadas como um valor booleano, verdadeiro ou falso. Uma instrução de controle de fluxo decide o que fazer com base em se sua condição é True ou False, e quase todas as instruções de controle de fluxo usam uma condição.

    print('Hello')

10. What keys can you press if your program is stuck in an infinite loop?

11. What is the difference between break and continue?

O break sai imediatamente da cláusula do loop while.

O continue volta para o início do loop e reavalia a condição do loop. (Isso também acontece quando a execução chega ao final do loop.)

12. What is the difference between range(10), range(0, 10), and range(0, 10, 1) in a for loop?

range(10) e range(0, 10) -> imprimem 1 sequencia de 10 números, começando em 0.

range(0, 10, 1) -> imprime 1 sequencia de 10 números, começando em 0 e de 1 em 1.
